# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly using
[GitHub Security Advisories](https://github.com/redhat-community-ai-tools/dci-mcp-server/security/advisories/new).

Please do **not** open a public issue for security vulnerabilities.

## Response

We aim to acknowledge reports within 5 business days and to provide a fix or
mitigation plan within 30 days, depending on severity.
